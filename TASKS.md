Standard Development Tasks

Task: init_repo
Purpose: initialize repository structure

Steps:

1. create README.md
2. create modules
3. add placeholder code
4. generate architecture documentation

Task: integrate_modules
Purpose: connect simulation systems

Steps:

1. analyze module APIs
2. connect simulation outputs
3. update architecture docs

Task: audit_tokens
Purpose: measure token efficiency

Steps:

1. analyze commit history
2. estimate token usage
3. compute token savings
