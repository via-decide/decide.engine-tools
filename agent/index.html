This page embeds the production-ready ViaDecide Agent widget via /js/viadecide-agent.js. Drop the same script tag into any ViaDecide page — no extra markup needed.

What’s included

  • Duplicate-injection guard via window.VDAgentLoaded.
  • Multi-turn chat — full conversation history sent to Gemini on every message.
  • Tasks tab — add, complete, and delete action items stored in localStorage.
  • Internal route intelligence — suggests clickable ViaDecide pages based on your question.
  • Lazy DOCX export — document library loads only when you click export.
  • Fresh page context on every send: pathname, title, and meta description.
  • All interactions bound via addEventListener — no global onclick attributes.

Quick test

1

Open the floating ✦ button in the bottom-right corner.

2

Go to Setup, paste a Gemini API key, and click Test key.

3

Go to Tasks, click + Add, type a task, and press Save.

4

Click ⬇ Export .docx to download your session.

The widget injects its own scoped DOM and CSS — works on any page without markup conflicts.
