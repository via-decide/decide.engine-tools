# Highway V2I Research Output

This folder stores generated research artifacts for the Highway-V2I lab module:

- simulation datasets (JSON snapshots)
- protocol performance logs
- protocol comparison tables
- infrastructure optimization reports

Runtime output can be exported through `HighwayExperimentRunner` as JSON and CSV.

Infrastructure Evolution Lab exports include:
- optimized RSU placement recommendations
- sensor layout recommendations (density, placement, camera/sensor angles)
- traffic control strategy recommendations (lane width, emergency lane priority, drainage slope)
