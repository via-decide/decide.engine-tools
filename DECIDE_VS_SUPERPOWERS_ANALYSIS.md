## 🎯 **What Each Does**

### **decide.engine-tools**
A **pattern & metadata library** for AI development. Provides:
- 40+ pre-built standalone tools (code generators, analyzers, research tools, games, business tools)
- Reusable architectural patterns and Firebase schemas
- Navigation registry and deployment templates
- Token economy optimization through pattern reuse
- Browser-native, zero-build-step implementation

**Core Value**: Save tokens by reusing patterns instead of researching/rebuilding from scratch.

### **Superpowers**
A **development methodology & skills framework** for AI agents. Provides:
- 7-stage workflow (brainstorm → plan → implement → test → review → merge)
- Structured process that creates a "contract between human and agent"
- Git worktree isolation
- Test-driven development enforcement
- Skills library for specific techniques
- Integration with Claude Code, Cursor, Codex, Gemini

**Core Value**: Save tokens and improve quality by using disciplined process, not rushing to code.

## ⚠️ **Weaknesses**

### **decide.engine-tools Weaknesses**

❌ **Early stage** — Just launched, smaller community (yet)
❌ **Browser-only** — Can't use for backend/DevOps/infra work
❌ **Pattern-dependent** — Benefits only if patterns exist for your use case
❌ **No process discipline** — Focus on tools, not methodology
❌ **Limited scope** — Specific to JavaScript/frontend initially
❌ **Community small** — 43K star gap vs Superpowers
❌ **Metadata burden** — Requires documenting patterns to compound
❌ **One-size-fits-most** — Patterns may not match every project

### **Superpowers Weaknesses**

❌ **Workflow overhead** — 7 stages add latency (spec → plan → code)
❌ **Adoption friction** — Requires plugin/integration setup
❌ **Process bookkeeping** — Planning docs are additional overhead
❌ **Not a toolkit** — Doesn't solve *what* to build, just *how* to build it
❌ **Test-driven mandate** — Can slow exploratory/prototype work
❌ **Methodology-first** — If you just want to code fast, it's slower
❌ **Learning curve** — Teams need to learn the 7-stage workflow
❌ **No pre-built solutions** — You still have to write every feature
