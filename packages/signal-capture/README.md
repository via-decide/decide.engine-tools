# Signal Capture

Signal Capture records embodied interaction rhythm as computational material: movement texture, hesitation, cadence, and physicality.

## Modules
- input sampler at fixed capture rhythm
- timing analyzer (pauses, intervals, cadence)
- fingerprint generation
- visualization datasets (heatmap, momentum, cadence)
- replay-engine integration for exact playback
- comparison delta descriptors (non-ranking)
- living-archive integration for preservation

## Principle
No engagement analytics or best-score systems. Preserve interaction personality.
