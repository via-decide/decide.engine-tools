# Constraint Lab

Constraint Lab is a deterministic creative experimentation environment for discovering computational identity.

## Core modules
- Runtime constraint chaining and toggles
- Constraint set: monochrome, tile quantization, delayed input, frame skip
- Experiment runner with replay bridge integration
- Side-by-side comparison view controls
- Artifact and notes capture helpers
- Aesthetic presets: gameboy, playdate, demoscene

## Principle
No scoring/ranking/coherence metrics. Human judgment and preserved artifacts drive discovery.
