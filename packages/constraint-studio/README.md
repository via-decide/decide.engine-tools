# Constraint Studio

Constraint Studio is a mutation-first laboratory for rapidly generating and comparing computational identity variants.

## Capabilities
- live constraint runtime + layer composition
- preset identities (gameboy, playdate, arcade1985, terminal, demoscene)
- mutation engine and variant tree generation
- replay-engine variant playback hooks
- signal-capture feel comparison hooks
- snapshot generation and living-archive integration

## Principle
No best-variant scoring. Preserve weird, memorable mutations and failed experiments.
